## ğŸš€ About Telephoto

**Telephoto** is a lightweight, open-source alternative to Google Photos that uses Telegram as a cloud storage backend. Built with Flutter, it provides a seamless experience for backing up your precious memories without worrying about storage limits or privacy concerns.

Unlike traditional cloud photo services, Telephoto leverages Telegram's generous file storage capabilities, allowing you to:
- Store unlimited photos in your personal Telegram account
- Access your photos from any device with Telegram
- Maintain full control over your data
- No third-party servers or storage services
